# 🎯 Adaptive Access AI - Hackathon Demo

AI-powered motor accessibility layer using Hermes function calling

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start the server (in one terminal)
npm run dev:server

# Load extension in Chrome
1. Open chrome://extensions
2. Enable Developer Mode
3. Click "Load unpacked"
4. Select the 'extension' folder

# Open demo page
npm run demo
```

## 🏗️ Architecture

```
adaptive-access-ai/
├── server/          # Hermes API integration
├── extension/       # Chrome extension
└── demo/           # Interactive demo
```

## 💡 Key Features

1. **Precision Cursor Control** - Dynamic sensitivity adjustment
2. **Magnetic Snap** - Automatic pull to targets
3. **Target Enlargement** - AI-detected UI adaptations
4. **Tremor Filtering** - Real-time movement stabilization

## 🎮 Demo Flow

1. Open demo page
2. Click "Simulate Tremor" to add motor difficulty
3. Try clicking tiny buttons (observe miss-clicks)
4. Click "Enable AI Assist" to activate Hermes
5. Watch as targets automatically enlarge and cursor stabilizes

## 🔑 Hermes Functions

- `adjustCursorSensitivity()` - Modifies mouse physics
- `enlargeClickTarget()` - Increases clickable areas
- `addMagneticSnap()` - Adds gravitational pull
- `detectAccessibilityNeed()` - Analyzes user patterns
- `simplifyInteraction()` - Converts complex to simple
