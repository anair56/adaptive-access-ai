# TremorSense Logo

Please save the TremorSense logo image here as `tremorsense-logo.png`

The logo should be the image with:
- "TremorSense" text where "Tremor" is in deep red (#8B2635) and "Sense" is in black
- The hand with circuit/tech design on the left side
- Transparent or white background
- Recommended size: 400x150px or similar aspect ratio

The logo will be displayed in:
1. Extension popup header
2. AI sidebar header
3. Any other branded areas

If the image is not found, the extension will show a text-based fallback.
